Detailed Description

MHA prepare configuration structure.

This structure contains information about channel number and domain for input and output signals of a MHA Plugin. Each plugin can change any of these parameters, e.g. by resampling of the signal. The only limitation is that the callback frequency is fixed (except for the plugins db and dbasync).
