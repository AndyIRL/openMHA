mha_spec_t *  s_spec,
unsigned int  ch 
)
private

Arrange the order of an fftw spectrum to the internal order.

The fftw spectrum is arranged [r0 r1 r2 ... rn-1 in in-1 ... i1], while the interal order is [r0 – r1 i1 r2 i2 ... rn-1 in-1 rn –].
