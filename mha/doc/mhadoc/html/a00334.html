Detailed Description

Name space for the MHA-Parser configuration language.

This namespace contains all classes which are needed for the implementation of the MHA configuration language. For details on the script language itself please see section The MHA configuration language.

List of valid MHAParser items

  • Sub-parser: parser_t
  • Variables:
    Numeric variables: int_t, vint_t, float_t, vfloat_t, mfloat_t
    Other variables: string_t, vstring_t, kw_t, bool_t
  • Monitors:
    Numeric monitors: int_mon_t, vint_mon_t, float_mon_t, vfloat_mon_t
    mfloat_mon_t
    mcomplex_mon_t
    Other monitors: bool_mon_t, string_mon_t, vstring_mon_t

Members can be inserted into the configuration namespace by using MHAParser::insert_item() or the insert_member() macro.
