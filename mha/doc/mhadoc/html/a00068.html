Detailed Description

Error reporting exception class.

This class is used for error handling in the MHA. It is used by the MHA kernel and by the MHA toolbox library. Please note that exceptions should not be used accross ANSI-C interfaces. It is necessary to catch exceptions within the library.

The MHA_Error class holds source file name, line number and an error message.
