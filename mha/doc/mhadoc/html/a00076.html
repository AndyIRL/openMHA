Detailed Description

Spectrum signal structure.

This structure contains the short time fourier transform output of the windowed input signal. The member num_frames describes the number of frequency bins in each channel. For an even FFT length , this is . With odd FFT lengths, it is . The imaginary part of the first bin is zero. For even FFT lengths, also the imaginary part at the Nyquist frequency is zero.

Data order of FFT spectrum.
